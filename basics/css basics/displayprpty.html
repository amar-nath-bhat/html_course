Block Element

  1. Go to the new line
  2. Occupy entire wifth from left to right
  3. We can apply custom width/heihgt
  4. Margin/Padding can be applied

Inline Element

  1. Donot go to the new line
  2. Occupy width of the content
  3. We cannot apply custom width/heihgt
  4. Margin/Padding can be applied only in horizontal direction and not in vertical
